### Automatic data abstraction

This repository contains the data and codes used for assessing the automatic data 
abstraction pipeline.

<a href="/data">/data</a> contains all the data sets used.

<a href="/pdfs">/pdfs</a> contains all the public literature on anopheles funestus that was filtered for this study.

The cleanned automactically extracted data can be found in <a href="data/clean_data_v2.csv">data/clean_data_v2.csv</a>

The manually extracted data derived from the <a href="https://vectoratlas.icipe.org" target="_blank">vector atlas platform</a> can be found in <a href="data/filteredVAData.csv">data/filteredVAData.csv</a>
